# Contributing to OC–REDCap Randomizer

Thank you for considering contributing! Contributions are welcome and encouraged.

## 🧩 Ways to Contribute
- Reporting bugs
- Submitting feature requests
- Improving documentation
- Submitting pull requests with enhancements or fixes
- Reviewing open pull requests

## 📦 Development Workflow

### 1. Fork the Repository
Click **Fork** on GitHub to create your own copy.

### 2. Clone Your Fork
```bash
git clone https://github.com/<your-username>/oc-redcap-randomizer.git
cd oc-redcap-randomizer
